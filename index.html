Generate RSA format SSH Key for Server Authentication

  1. Open Terminal (MacOS/Linux) or Command Line/Powershell/Git Bash (Windows)

  2. Run this command below:

    3. > ssh-keygen -t rsa


  3. When you're prompted to "Enter a file in which to save the key", you can press Enter to accept the default file location. Please note that if you created SSH keys previously, ssh-keygen may ask you to rewrite another key, in which case we recommend creating a custom-named SSH key. To do so, type the default file location and replace id_CUSTOM with your custom key name (like id_server).

    4. > Enter file in which to save the key (/c/Users/YOU/.ssh/id_CUSTOM):


  4. At the final prompt, type a secure passphrase but keep it simple so that you don't forget it.

  5. Now using the File Explorer (Windows) / Finder (MacOS), locate the genereated file as mentioned in step-3. like: id_server.pub
